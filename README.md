##  Overview

**CARVAN** is a modern automotive marketplace web app showcasing a premium inventory of vehicles across multiple categories. The platform features new arrivals, a pre-owned section, brand partnerships, and a sleek UI designed for an elevated browsing experience.

---

##  Features

-  **The Range** — Browse vehicles by category: SUVs, Sports, Vans, Sedans, Jeeps, Vintage & Compact
-  **Featured Arrivals** — Curated listings with price, mileage, location & transmission details
-  **Pre-Owned Section** — Certified pre-owned vehicles with full specifications
-  **50+ Brands** — Partnerships with Ferrari, Toyota, Audi, BMW and more
-  **Test Drive Booking** — Users can book a test drive directly from the site
-  **Contact & Inquiry Portal** — Dedicated sales and support contact page
-  **24/7 Encrypted Live Support**
-  **Newsletter / Frequency Feed** — Subscribe for live inventory updates

### Deployment

The project is deployed on **Netlify** via continuous deployment from GitHub.

To deploy your own copy:
1. Fork this repo
2. Connect it to your Netlify account
3. Set the publish directory to the project root (or `dist/` if using a build step)
4. Deploy
